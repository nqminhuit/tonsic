Tonsic — Chord Trainer (static Next.js)

Quick start:

1. Install dependencies: npm install
2. Run dev server: npm run dev

Notes:
- The app uses the Web MIDI API to read MIDI note-on messages from a connected keyboard.
- On Android, Web MIDI support varies by browser and may require special configuration; an on-screen keyboard is provided as a fallback for testing.
- Connect your USB MIDI keyboard to your PC or Android device and grant MIDI permission when prompted.

This repository contains a minimal scaffold: pages/, components/, and lib/ for chord logic.
